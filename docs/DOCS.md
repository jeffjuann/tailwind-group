# Documentation

### Group custom modifier
```ts
twOn('hover', 'text-red-500 font-bold') // hover:text-red-500 hover:font-bold
twOn('hover', 'text-red-500 font-bold', 'bg-yellow-500') // hover:text-red-500 hover:font-bold hover:bg-yellow-500
```
